##1.APP接口简介

###1.1.APP接口介绍

  • PHP面向对象接口:抽象类，需要interface关键字定义

    /**
 * 定义一个接口
 * 提供一个标准
 */
    interface video{
        public function getVideos();//抽象方法1
        public function getCount();//抽象方法2
    }

/**
 * Class movie
 * 必须实现video的所有抽象方法
 */
    class movie implements video{
        public function getVideos()
        {
            echo 'getVideos';
        }

        public function getCount()
        {
        echo 'getCount';
        }
    }
  • APP接口(也称通信接口):由接口地址、接口文件（.php文件，负责处理业务逻辑）和接口数据组成.

###1.3.通信格式区别

  • XML格式:XML(Extensible Markup Language),扩展标记语言，可以用来标记数据、定义数据类型，允许用户对标记语言进行定义的源语言。
    XML节点可以自定义；HTML标签不可以自定义。

    /**
*有且只能有一个根节点,如"<item></item>"
*/
<?xml version="1.0" encoding="UTF-8"?>
<item>
   <title>title</title>
   <test id="1"/>
   <adress>foshan</adress>
   </item>

  • JSON格式:JSON(JavaScript Object Notation),一种轻量级的数据交换格式。

    {
      "errcode": 0,
      "errmsg": "ok",
      "data": {
                "id": 1,
                "title": "cat",
                "api_version": "0.1.0",
                "release_date": "2015-07-30",
                "time": 1438678412
              }
}

  • 对比:

    可读性：xml>json<br />
生成速度：json>xml<br />
传输速度:json>xml<br />
